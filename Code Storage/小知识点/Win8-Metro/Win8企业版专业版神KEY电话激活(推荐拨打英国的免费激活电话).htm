Win8企业版专业版神KEY电话激活(推荐拨打英国的免费激活电话)-2012年8月18日激活成功

(2012-08-19 00:05:28)
转载▼
标签：

win8

企业版

专业版

神key电话激活

 分类： 电脑知识
昨晚激活了笔记本的，今天下午一口气把家里的两台台式机电脑都激活了，果然有些序列号已经失效了，激活第三台电脑时费了好几个小时。。
-----------------------------------------------------------------------------------------------------
win8企业版的孩子，昨晚还没激活的孩子抓紧额（专业版的也可以  自己换key啊）
1、win + x 弹出如下菜单，选择红框所示选项：


2、输入如下命令替换激活密匙：
slmgr /ipk BYXV8-9YN72-VTF7H-8G7PT-WW8X2 (昨晚可用，已失效)
slmgr /ipk PF9TR-KNB98-T7JPF-TWF8W-F3G7C （没成功）
slmgr /ipk DWV49-3GN3Q-4XMT7-QR9FQ-KKT67 （下午可用）
slmgr /ipk NB7P3-C66T2-848FB-M3BJB-2PKVC （下午可用）
slmgr /ipk JQN3D-D9DMG-97X94-K2QW7-4CWXV
slmgr /ipk KRRY8-76NW2-FFTXH-KPKM9-CDG67
（此密钥也可以激活Pro版）
     


3、拔掉网线，或者禁止掉本地连接后点击“激活”按钮：


4、出现如下错误提示：


5、错误后点关闭自动跳出下面的电话激活       选择“通过电话激活”按钮：


6、选择“中国”： 如果拨打英国的免费电话，这里选英国



7、开始打电话激活系统：（纯电脑姐姐  别怕人工服务的烦恼）


同志们，安装SKYPE，打英国客服吧+44 800 018 8354用Skype打，按1，再按1，输入号码，俺1，记下A到H，#下一个，*重新报。
我选择了英国的免费激活电话：+44 800 018 8354，无充值的Skype是能打通的，然后我就成功激活了
前面下载注册Skype 就不说了，注册很简单也不需要邮箱验证神马的。
打电话免费不需要积分也不需要点数

拨打电话是选择地区英国然后就输入客服电话： 8000188354

进入接通之后，打开Skype小数字键盘直接按1，然后语音停顿一下就可以进入下一步了，没停顿就接着按下1，下一步就直接输入9组数字了。输入数字过程中输入错误的话，不要紧语音提示会让你重新输入那一组数字。9组数字输完之后，一般会有3种提示：
一种是转入客服，果断挂电话，重新找密钥去
二种是不能激活，让你去官网查询，她挂了你的电话，还得重新再找密钥去
三种声音比较幸福点，在同一台电脑上安装按1，在其他电脑上安装按2，千万不能按2，按了2你还得重新找密钥去，这个密钥就费了。
好了按了1就简单了，按1之后语音有一大堆废话，不听，按#，就开始报A~H的数字了，一般每组报两遍，只喜欢听一遍的话就按#，最后一组H报完之后可以按1重新听一遍。不听就直接挂电话激活吧。
PS：
可以把KEY和得到的激活编号保存下来，以后每次装系统的时候如上面的操作后直接输入即可激活（不需要打电话了）。
由于微软服务器已经存在了你的激活信息，所以下次重装的时候可能输入KEY后直接立即激活。
有图有真相




台湾电话+886800008833【接通后，听到机器人说话就按 2，稍等会，才会转到微软注册中心，然后按 1  】
英国电话+448000188354【接通后，听到机器人说话就按 1 ，然后继续 1，听到first这个单词，你就要准备输入了】
  等他反馈的48位数字 保存下来 输入电脑 就可以永久激活了  这48位数字留到以后 重装也可以激活的 完全可行！

  最后大家 用 slmgr.vbs -xpr   和  slmgr.vbs -dlv 验证下吧  看到永久这两字我得意地笑  ！

 
最近更新
用电话激活机器会出现几串数字 有人说这数字会更变 变的话 48位激活码就无法再次激活  

所以这次我再一次试验  再不更改硬件情况下  更改系统时间  48位激活码一样能用   现在激活得到激活码 可以不限时间激活
